# 🎮 Tic Tac Toe in Java

A simple **console-based Tic Tac Toe game** built in Java.  
This project demonstrates how fundamental programming concepts can be applied to create interactive applications.

---

## ✨ Features
- Two-player mode (X vs O)
- 3x3 grid using a **2D array**
- Input validation (no overwriting moves)
- Win detection across rows, columns, and diagonals
- Draw detection when the board is full
- Clean and easy-to-read code structure
